# Suggested Commands

## Build & Run
- `npm install` - Install dependencies
- `npm run build` - Compile TypeScript to dist/
- `npm run serve` - Start HTTP server on port 8000
- `npm start` - Build + serve (combined)
- `npm run dev` - Watch mode development

## Docker
- `docker build -t mandelbrot-explorer .` - Build Docker image
- `docker run -p 8000:8000 mandelbrot-explorer` - Run container

## System Utils (Darwin)
- `git`, `ls`, `cd`, `grep`, `find` - Standard unix tools
